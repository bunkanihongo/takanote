# 📝 たかのーと — 日本語学習ノート

Chirpy Jekyll テーマで作られた日本語学習ブログです。

## 🌐 サイト

[https://bunkanihongo.github.io/takanote](https://bunkanihongo.github.io/takanote)

## 📖 コンテンツ

- **文法** — 助詞、て形、敬語など
- **単語** — N5/N4 単語リスト、日常会話フレーズ
- **読解** — ショートストーリー、やさしいニュース

## 🛠 技術

- [Jekyll](https://jekyllrb.com/) + [Chirpy Theme](https://github.com/cotes2020/jekyll-theme-chirpy)
- GitHub Pages + GitHub Actions で自動デプロイ

## ✍️ 投稿の追加方法

`_posts/` ディレクトリに `YYYY-MM-DD-title.md` の形式で Markdown ファイルを追加するだけ。

```markdown
---
title: 記事のタイトル
date: 2026-07-01 00:00:00 +0900
categories: [文法]
tags: [タグ1, タグ2]
---

ここに本文を書く
```
